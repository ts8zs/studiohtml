# 安装与运行

# 点击这里下载 MiniStudio 最新安装包open in new window

  1. 解压缩下载的 MiniStudio 安装包open in new window

  2. 双击 MiniStudioAutoSetup.exe 等待几分钟。

  3. 安装好了会自动运行！

  4. 以后可以双击桌面上的 ministudio.lnk 进入 MiniStudio

MiniStudioAutoSetup.exe为自动安装程序，安装路径为C:\ProgramData

安装完成会在会在桌面上生成ministudio.lnk快捷方式，并且会自动运行解压目录下的MiniStudio.exe

注意：安装过程需要一点时间，不要重复点击。
