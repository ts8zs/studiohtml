# 美术工具导览

在沙盒工具的蓝图微缩或者骨骼模型分类中 在搜索栏下方点击右键，在菜单中选择新建。选择新建，即可以新建出相应的资源文件，双击即可编辑。在新建的资源文件上又见几个上传的时期称为云端，资源云端资源就可以使用于游戏之中。

蓝图是一种快速构建方块场景的一种方式，我们可以将预先构建好的方块组合放置在场景之中。 微缩是通过方块来构建体素模型的一种方式我们可以通过微缩来创建一些体素模型来放置在我们的场景之中。 骨骼模型是一种迷你世界的一种高级模型组织方式。通过将多个模型来拼组成一个复杂模型，而骨骼模型中的子模型具有。移动，旋转和缩放。等基本操作 通过组合这些操作和时间轴。我们可以为骨骼模型制作动画。
