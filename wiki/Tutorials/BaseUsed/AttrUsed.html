# 属性栏的使用

当双击选中场景窗口中节点对象时，属性栏会展示出该节点相关的属性，我们可以对属性进行编辑操作，如位置，颜色等。

  • 点击非灰色的属性可以编辑当前属性
  • 点击分类或属性前方的箭头可以展开或折叠属性。

# 节点属性的编辑

节点属性包含： 基础属性 和 自定义属性

# 基础属性

  • 基础属性是系统内置的，不允许删除，但绝大部分展示出来的属性是可以被开发者修改的。
属性类型说明
数字可以输入数字
字符串可以输入文字
bool可以勾选确认
数组可以在展开后输入多组相同类型值 未展开时编辑暂时无效
颜色类似数组类型 未展开时编辑暂时无效
资源可以直接输入路径 点击后再次点击后方的三个点 可以选择上传资源或使用本地资源
选项可以从提供的多个选项中选择一个

# 自定义属性

  • 当某个节点对象的属性不能满足开发者的需求的时候，我们提供了扩展属性，这个时候开发者可以通过 属性窗口 最下方的 Custom Property 按钮添加自定义属性

支持扩展的自定义属性类型有

属性类型描述
boolBOOL类型
colorRGB颜色
color arrayRGB颜色列阵（多个RGB颜色）
double数字类型
string字符串类型
vector2二维坐标
vector3三维坐标
vector4四维坐标
