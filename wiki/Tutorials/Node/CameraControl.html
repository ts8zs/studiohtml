# 相机的控制

MiniStudio提供数种用于配置游戏镜头的内置选项，以及编写自定义镜头系统脚本的功能。

# 内置设置

常见的镜头设置可在 MiniStudio 中直接配置：

在 场景 窗口中，选择StarterPlayer对象。

在Properties（属性）窗口中，向下滚动并找到Camera（镜头）部分。利用这些选项，您可以轻松地配置您的游戏的镜头。

# 缩放距离

运行模式下，CameraMaxZoomDistance 和 CameraMinZoomDistance 共同设置玩家可以相对其角色缩放镜头的距离。如果设置非常高的值，例如 500，玩家甚至将可以将镜头缩放到太空中。

如果要将镜头锁定在离玩家固定的距离上，请将这两个属性设为相同的值。

# 镜头模式

CameraMode 属性设置镜头的整体行为：

设置描述
Classic经典的 MiniStudio 镜头允许玩家放大与缩小（除非镜头已锁定），并围绕玩家旋转镜头。
LockFirstPerson锁定镜头为第一人称模式。在此模式下，玩家角色的所有部件/元素对玩家都不可见，除了玩家所装备的工具。
